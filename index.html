Firmware Implementation

Key parts of my Arduino code with explanation

bool detectSlip(long pressure) { bool slip = false; if (pressure < MIN_PRESSURE) slip = true; if ((lastPressure - pressure) > DROP_THRESHOLD) slip = true; if (pressure > MIN_PRESSURE) lastStepTime = millis(); else if (millis() - lastStepTime > STEP_TIMEOUT) slip = true; lastPressure = pressure; return slip; }

üß† Slip Detection Logic

Slip is detected using three independent conditions to reduce false positives.

  • Minimum pressure threshold
  • Sudden pressure drop
  • Timeout without stable step
